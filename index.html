A simple FontAwesome like kit for Twitch Badges, Bits and More.

Twitchcons was a library originally used by TwitchOS, a failed concept for bringing streaming to it's own basic operating system that was more of an extension of Windows. We planned on working on it's concept but after some work and not wanting to step on the toes of Twitch Interactive, we cancelled the project and started working on a solo library version of the icon system we used within the OS.

Below are some examples setup and basic usage.

Setup

Setting up twitchcons in any environment is simple with the automatic update system to send updates as we make changes. Simply apply our CSS to your project via a link tag to have access to our complete system.

Example: <link rel="stylesheet" href="https://vysum.github.io/twitchcons/css/twitchcons.min.css">
